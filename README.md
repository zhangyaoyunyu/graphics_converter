# graphics_converter



## 概述

本工具是将Bentley系列软件的dgn文件图形转换为web支持的三维数据格式

## 主要功能点

- [ ] 支持格式gltf,3dtiles
- [ ] 支持规则格网空间索引
- [ ] 支持投影变换

## 使用简要说明
1.MDL Applications中，选择CEPrj.ma文件并载入;
2.key-ins中，输入atcditools->export->3dtiles，打开转换程序界面;
```
ps:后续逐步增加
```
